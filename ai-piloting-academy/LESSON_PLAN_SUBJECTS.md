# AI Piloting Academy — Session Topics

Pick 5 of these for Sessions 2-6. Topics build progressively from daily use to advanced workflows.

---

## Session 2: "Your Daily Co-Pilot"
**Theme:** Making Claude Code your default tool for everyday tasks
- Writing & editing code with Claude in the terminal
- Navigating codebases — "read this file," "find where X is defined"
- Quick wins: generating commit messages, explaining unfamiliar code, writing tests
- **Homework:** Use Claude Code for at least 3 real work tasks this week. Log what worked and what didn't.

## Session 3: "Prompting Like a Pilot"
**Theme:** The art of effective prompting — from vague to precise
- Anatomy of a great prompt: context, constraints, examples, output format
- CLAUDE.md as "persistent prompts" — setting defaults so you prompt less
- Prompt patterns: few-shot, chain-of-thought, role-setting
- **Homework:** Write a CLAUDE.md for your main project. Include coding style, review preferences, and team conventions.

## Session 4: "The Skill Workshop"
**Theme:** Building and sharing custom skills across the team
- Live skill-building session — everyone builds one together
- Skill design patterns: when to use frontmatter options, supporting files, argument hints
- Sharing skills: project-level skills committed to git for the whole team
- **Homework:** Create a team skill — one that helps anyone on the product team, not just you.

## Session 5: "Agents & Multi-Step Workflows"
**Theme:** Going beyond single prompts to autonomous task execution
- Understanding Claude Code's agent mode — sub-agents, parallel execution
- The Agent tool: when and how to delegate to specialized sub-agents
- Building agent orchestration in CLAUDE.md (planner, code-reviewer, tdd-guide patterns)
- **Homework:** Set up an agent workflow for a real task — code review, bug investigation, or feature planning.

## Session 6: "AI-Powered Product Work"
**Theme:** Claude beyond code — for PMs, designers, and marketers
- Writing PRDs and specs with Claude (structured prompting for product docs)
- Design critique and accessibility review workflows
- Email marketing: generating variants, A/B test copy, subject line optimization
- Competitive analysis and market research with web search tools
- **Homework:** Use Claude to produce a real deliverable for your current sprint.

## Session 7: "MCP Servers & External Tools"
**Theme:** Connecting Claude to your entire toolchain
- What are MCP servers? Architecture and how they work
- Connecting to Slack, ClickUp, Google Calendar, Gmail, Supabase
- Building read/write workflows across tools (e.g., "summarize this Slack thread into a ClickUp task")
- **Homework:** Connect one MCP server to your Claude setup and build a workflow around it.

## Session 8: "Testing & Quality with AI"
**Theme:** Using Claude to write better tests and catch bugs
- TDD workflow with Claude: red-green-refactor
- Generating unit, integration, and E2E tests
- Code review automation — using the code-reviewer agent pattern
- Security scanning with the security-reviewer agent
- **Homework:** Add test coverage to an untested module in your project using TDD with Claude.

## Session 9: "Advanced Patterns & Automation"
**Theme:** Power-user workflows for maximum leverage
- Hooks system: PreToolUse, PostToolUse, auto-formatting
- Batch operations with /batch for large-scale changes
- Git workflow automation: conventional commits, PR creation, changelog generation
- Custom agent creation for your team's specific needs
- **Homework:** Build a hook or automation that saves you 15+ minutes per week.

## Session 10: "The AI-Native Team"
**Theme:** Capstone — building a culture of AI-first work
- Team retrospective: what changed, what stuck, what surprised you
- Sharing top skills and workflows across the team
- Building a shared skill library for the product org
- Setting up onboarding: how new team members get productive with AI fast
- **Final project:** Each person presents their best AI workflow to the group.

---

## Recommended 5-Session Sequence

| Session | Topic | Why This Order |
|---------|-------|----------------|
| 2 | Your Daily Co-Pilot | Get everyone using it for real work immediately |
| 3 | Prompting Like a Pilot | Build the skill that makes everything else better |
| 4 | The Skill Workshop | Team collaboration + reusable automation |
| 5 | AI-Powered Product Work | Role-specific value for PMs, design, marketing |
| 6 | MCP Servers & External Tools | Connect Claude to the full toolchain |

Sessions 7-10 are great follow-ups if the team wants to continue beyond the initial 6.
